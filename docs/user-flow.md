## 1. 온보딩 — 최초 설정

앱을 처음 실행하면 4단계 온보딩을 거칩니다. 각 페이지에 Lottie 애니메이션을 배치해 딱딱하지 않게 설정을 안내합니다.
<video src="https://github.com/user-attachments/assets/0a4f5a20-84c7-4a9b-8e43-be2aa9971536" controls width="300"></video>

### Page 0 — 닉네임 입력

닉네임을 입력하면 홈 화면 인사 문구와 설정 프로필에 표시됩니다. 한국어·영어만 허용하고 최대 10자로 제한합니다. 닉네임이 비어 있으면 다음 버튼이 비활성화됩니다.

### Page 1 — Health Connect 권한 요청

걸음 수 읽기 권한을 요청합니다. 거부해도 앱은 계속 사용할 수 있고, 홈 화면에서 다시 요청할 수 있습니다.

### Page 2 — 일일 목표 걸음 수 설정

슬라이더로 하루 목표 걸음 수를 설정합니다. 범위는 5,000~20,000보이며 기본값은 10,000보입니다.

### Page 3 — 알림 설정

피크아워 기반 걷기 알림 수신 여부를 설정합니다.

## 2. 홈 화면 — 오늘의 걷기 현황

<video src="https://github.com/user-attachments/assets/f0b097c2-709b-491f-bc81-533d4e68211b" controls width="300"></video>

홈 화면은 오늘의 걷기 상태를 한눈에 보여주는 대시보드입니다.

**구성 요소:**

- **닉네임 인사**: "OO님, 오늘도 걸어요!" — 설정한 닉네임이 반영됩니다.
- **걸음 수 링**: 오늘 걸음 수와 목표 달성률을 원형 링으로 표시합니다. 걷는 중으로 감지되면 링이 pulse 애니메이션으로 강조됩니다.
- **스트릭 뱃지**: 연속 목표 달성 일수를 표시합니다.
- **날씨 카드**: 현재 기온, 날씨 상태, 오늘 걷기 적합 여부를 안내합니다.
- **지난주 리포트 요약**: 지난주 총 걸음 수와 달성률 카드를 홈에서 바로 확인할 수 있습니다.
- **예보 배너**: 오늘 피크아워 예보를 표시하고, 탭하면 예보 상세 BottomSheet가 열립니다.

**Health Connect 미연결 상태 처리:**

| 상태 | 표시 내용 |
|---|---|
| 권한 미허용 | 권한 요청 카드 |
| HC 미설치 기기 | HC 미지원 안내 카드 |
| 정상 | 전체 대시보드 |

**미션 달성 자동 감지:**

걸음 수가 목표치에 도달하면 자동으로 포인트가 지급됩니다. 하루에 한 번만 지급되며 앱을 재시작해도 중복 지급되지 않습니다.

## 3. 미션 화면 — 오늘의 목표

<video src="https://github.com/user-attachments/assets/80615d15-a408-48ae-9da8-f76dd37a139a" controls width="300"></video>

오늘 달성해야 할 걸음 수 목표와 달성 현황을 카드로 보여줍니다.

**미션 타입:**

| 타입 | 조건 | 보상 |
|---|---|---|
| 일반 미션 | 매일 | 20 포인트 |
| 회복 미션 | 전날 미달성 시 | 20 포인트 |

전날 목표를 달성하지 못하면 다음 날 더 낮은 목표의 회복 미션이 제공됩니다. 회복 미션을 달성하면 스트릭이 유지되고, 실패하면 스트릭이 초기화됩니다.

## 4. 주간 리포트 — 아카이브 & 상세

<video src="https://github.com/user-attachments/assets/34f5408a-a072-4eb0-b41b-701801ab9d5b" controls width="300"></video>

최근 12주의 걷기 기록을 카드 목록으로 모아볼 수 있습니다.

**아카이브:**
- 주별 총 걸음 수, 달성률, 날짜 범위를 카드로 요약합니다.
- 현재 진행 중인 주는 잠금(LOCKED) 상태로 표시됩니다.

**상세:**
- 카드를 탭하면 요일별 막대 그래프와 주간 통계를 확인할 수 있습니다.
- "리포트 공유하기"를 탭하면 주간 요약 이미지를 캡처해 외부 앱으로 공유할 수 있습니다.

## 5. 월간 기록 — 달력 뷰

<video src="https://github.com/user-attachments/assets/de24b6d3-4fdb-45f9-b455-0a524b6437fd" controls width="300"></video>

한 달 단위 캘린더로 날짜별 걸음 수를 시각화합니다. 좌우 스와이프로 월을 이동할 수 있습니다.

**날짜 셀 상태:**

| 상태 | 시각 표현 |
|---|---|
| 달성 | 초록 원 |
| 오늘 (미달성) | 테두리 원 |
| 부분 달성 | 회색 원 |
| 미기록 | 투명 |

날짜를 탭하면 하단에 해당 날짜의 걸음 수 상세 패널이 슬라이드업됩니다.

## 6. 월간 리캡 — 스토리 요약

<video src="https://github.com/user-attachments/assets/dcd878f8-d4a2-4199-96fa-7bae0fe4d2d8" controls width="300"></video>

홈 화면의 리캡 카드를 탭하면 지난달 걷기 활동을 Instagram Stories 형식 8장 슬라이드로 요약해 보여줍니다.

**슬라이드 구성:**

| 슬라이드 | 내용 |
|---|---|
| 1 | 월 라벨 + 인트로 |
| 2 | 이달 총 걸음 수 |
| 3 | 하루 평균 걸음 수 |
| 4 | 소모 칼로리 추정 |
| 5 | 목표 달성일 + 달력 도트 시각화 |
| 6 | 가장 많이 걸은 날 |
| 7 | 최장 연속 달성 기록 |
| 8 | 걷기 페르소나 (평균 기반 분류) |

슬라이드는 5초마다 자동으로 넘어가고, 일시정지·재생·수동 탭 이동을 지원합니다.

## 7. 걷기 예보 — 피크아워 알림

<video src="https://github.com/user-attachments/assets/11b4d70f-3bdb-4ee3-87a6-ec3e9cf002aa" controls width="300"></video>

과거 7일간의 시간대별 걷기 패턴을 분석해 사용자가 가장 잘 걷는 시간대(피크아워)를 파악하고, 그 시간에 맞춰 알림을 보냅니다.

**예보 BottomSheet 정보:**
- 오늘 예상 피크아워
- 주간 걷기 추세 (증가/감소)
- 현재 스트릭 위험도

**알림 동작:** 피크아워 시간대에 알림이 발송되며, 알림을 탭하면 홈 화면으로 바로 이동합니다.

## 8. 설정 — 닉네임 & 포인트

<video src="https://github.com/user-attachments/assets/f7a02b39-bb57-486a-be53-4f94f23b3158" controls width="300"></video>

### 프로필 섹션

화면 상단에 이니셜 아바타, 닉네임, 포인트 뱃지를 표시합니다.

### 닉네임 수정

닉네임 영역을 탭하면 수정 BottomSheet가 슬라이드업됩니다. 저장하면 홈 화면 인사 문구와 프로필이 즉시 반영됩니다.

### 기타 설정

- 일일 목표 / 회복 걸음 수 조정
- 알림 켜기/끄기
- 라이트 / 다크 / 시스템 테마 즉시 전환

## 9. 홈 위젯

<video src="https://github.com/user-attachments/assets/a14c7482-abe4-4e06-8212-dd79b2c56d27" controls width="300"></video>

앱을 열지 않고도 오늘 걸음 수와 미션 진행 상태를 홈 화면에서 확인할 수 있습니다.

**표시 정보:**
- 오늘 걸음 수 / 목표 걸음 수
- 진행률 바
- 오늘 미션 제목

15분마다 자동으로 업데이트되며, 새로고침 버튼을 탭하면 즉시 최신 걸음 수로 동기화됩니다.

## 10. 리워드 — 보상 시스템 (예정)

<video src="https://github.com/user-attachments/assets/6212c039-a980-411a-a363-63c0b776fed1" controls width="300"></video>

바텀 내비게이션의 리워드 탭에서 접근합니다. 현재는 티저 화면으로, 앞으로 추가될 기능을 미리보기 카드로 안내합니다.

**현재 포인트 적립 기준:**

| 행동 | 포인트 | 중복 방지 |
|---|---|---|
| 일일 미션 달성 | +20 | 하루 1회 |
| 회복 미션 달성 | +20 | 하루 1회 |

**예정 기능:** 리워드 스토어, 뱃지 컬렉션, 레벨 시스템
